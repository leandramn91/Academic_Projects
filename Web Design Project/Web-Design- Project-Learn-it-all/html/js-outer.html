    JavaScript Quick Tips:

  1. Don’t forget var keyword when assigning a variable’s value for the first time.
  2. The use of semi-colons for line termination is a good practice.
  3. Be careful when using typeof, instanceof and constructor.
  4. Create a Self-calling Function.
  5. Use a switch/case statement instead of a series of if/else.
  6. Use switch/case statement with numeric ranges.
  7. Pass functions, not strings, to setTimeout() and setInterval().
